This API is used to detect an i2c device and on success return a positive handle.

Parameters:
[in]i2c_instancei2c device instance (0 or 1).
[in]i2c_device_typei2c device type (0: std 8b | 1: std 16b | 2: sccb 8b | 3: sccb 16b | 4: no subaddr)
[in]i2c_device_addri2c device slave address.
[in]chip_id_regNormally i2c slave device has a chip id register with a default value, which is typically used to detect the chip and its version. If chip_id_reg and chip_id_reg_value both are provided as zero then it will be treated as no i2c device detection is required.
[in]chip_id_reg_valuedefault value of chip id register..
Returns:
+ve handle on Success, -1 on Failure.
