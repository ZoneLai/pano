typedef void(* mxuvc_audio_cb_t)(unsigned char *buffer, unsigned int size, audio_format_t format, uint64_t ts, void *user_data, audio_params_t *param)

audio callback function to register with mxuvc audio for receiving audio capture data.

Parameters:
bufferpointer to the audio data received from the camera
sizesize of the audio data received from the camera
formatformat of the audio frame received
tsaudio timestamp in terms of ticks of 90khz clock where each tick corresponds to 1/(90 * 1000) sec or 1/90 ms
user_datapointer to user context/user data that the user set when registering the callback function. The pointer value is returned as given to MXUVC at the time of registering the callback function i.e. in mxuvc_audio_register_cb
parampointer to the audio parameters in the current frame received from the audio channel in the callback function
